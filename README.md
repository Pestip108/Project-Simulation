# Project-Simulation

A full Project Simulation


## Frontend

Use http-server:
```
http-server ./ -p 3000 -c-1 --cors
```

## Backend

If you haven't before, setup an enviroment variable for the encryption key:
```powershell
setx ENCRYPTION_KEY "your_key"
```

Run Server:
```
go run ./cmd/server
```
